# Command Reference

This directory contains detailed documentation for all Deposium CLI commands.

## Core Commands

- **[auth](auth.md)**: Manage API keys and authentication sessions.
- **[config](config.md)**: Manage local configuration settings.
- **[health](health.md)**: Check connectivity and status of Deposium services.
- **[logs](logs.md)**: View and search MCP server logs.

## Search & Discovery

- **[search](search.md)**: Primary document search (Vector, FTS, Fuzzy).
- **[graph](graph.md)**: Graph analysis and traversal.
- **[tools](tools.md)**: List available MCP tools.

## AI & Intelligence

- **[chat](chat.md)**: Interactive AI chat mode.
- **[compound](compound.md)**: Multi-tool reasoning and research.
- **[dspy](dspy.md)**: Intelligent query routing and optimization.
- **[intelligence](intelligence.md)**: Query analysis, suggestions, and summarization.
- **[leanrag](leanrag.md)**: Optimized retrieval and ranking.

## Data Management

- **[corpus](corpus.md)**: Corpus statistics and evaluation.
- **[duckdb](duckdb.md)**: DuckDB connection, federation, and mcp-server.
- **[upload-batch](upload-batch.md)**: Batch file upload utility.
- **[query-history](query-history.md)**: Track and analyze user query history.

## Evaluation & Metrics

- **[benchmark](benchmark.md)**: OpenBench LLM benchmarking.
- **[evaluate](evaluate.md)**: Metrics, dashboards, and feedback submission.

## Utilities

- **[mermaid](mermaid.md)**: Diagram generation and querying.
- **[ui](ui.md)**: Launch interactive dashboards.
